# NoCarry

NoCarry is a Software as a Service (SaaS) platform designed to facilitate fair grading of university group projects. With advanced features for automatic contribution tracking, freeloader detection, and AI-powered reports, NoCarry aims to enhance accountability and improve the educational experience for students and educators alike.

## Overview

In group projects, evaluating individual contributions can often be a challenge. NoCarry addresses this issue by providing tools that automatically track contributions, ensuring that every team member is fairly graded based on their actual workload and input. This platform promotes a culture of collaboration and responsibility among students.

## Features
- **Automatic Contribution Tracking**: Gain insights into who contributed what during group projects, allowing for transparent and fair evaluation.
- **Freeloader Detection**: Identify team members who are not contributing fairly to the project, helping professors take appropriate actions.
- **AI-Powered Reports**: Receive automated reports that summarize contributions and provide insights into team dynamics and performance.
- **User-Friendly Interface**: Navigate effortlessly through the platform with a clean and intuitive user interface.
- **Real-Time Collaboration**: Collaborate with team members in real-time, providing a seamless group work experience.

## Tech Stack
- **Frontend**: React.js for a dynamic user interface.
- **Backend**: Node.js with Express to handle API requests and serve data.
- **Database**: MongoDB for flexible data storage solutions.
- **AI & Machine Learning**: Integrated AI models for contribution analysis and report generation.

## Quick Start
1. **Sign Up**: Create an account on the NoCarry platform.
2. **Create a Project**: Start a new group project by inviting your teammates.
3. **Monitor Contributions**: Use the dashboard to view contributions as they happen.
4. **Generate Reports**: At the end of the project, generate detailed reports to review team contributions.

## Architecture
NoCarry operates on a microservices architecture, allowing for scalability and flexibility. The frontend communicates with the backend services via REST APIs. Our backend is designed to handle various aspects of user management, project tracking, and reporting, ensuring a smooth user experience. The integration of AI models enhances our reporting capabilities, making them smarter and more insightful.
